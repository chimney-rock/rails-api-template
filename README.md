# Rails API Template

```shell
rails new delete_me \
  -m https://raw.githubusercontent.com/chimney-rock/rails-api-template/master/template.rb \
  -d postgresql \
  --skip-webpack-install \
  --skip-action-cable \
  --skip-sprockets \
  --skip-javascript \
  --skip-turbolinks \
  --skip-test \
  --api
```
